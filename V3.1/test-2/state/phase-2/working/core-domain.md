# Core Domain — working file

## Core Domain: Pomiar czasu + Wyniki i ranking (BC-07 + BC-08)

**Uzasadnienie:**
System jest definiowany przez dwa nierozerwalnie powiązane konteksty:
- BC-07 Pomiar czasu — unikalna zdolność techniczna: integracja ze sprzętem (dyskietka + czytnik), dokładna rejestracja przejazdów w czasie rzeczywistym, obsługa dystansów wydłużonych
- BC-08 Wyniki i ranking — unikalna wartość biznesowa: klasyfikacja per dystans × płeć, obsługa kar pozycyjnych (nie czasowych), publikacja na żywo i końcowa

Żaden inny kontekst nie tworzy unikalnej przewagi — rejestracja, opłaty, grupy startowe to konteksty wspierające obecne w każdym systemie obsługi wydarzeń sportowych.
